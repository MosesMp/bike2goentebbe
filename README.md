# bike2goentebbeWelcome to the Bike Tour Uganda project! This is a minimalistic website created to help promote a small bike tour business based in Uganda. The website provides information about the tour packages offered, pricing, and contact information.

Getting Started These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.
